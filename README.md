# Newton Rate Interface

Newton Rate interface made with Next.js, React, Typescript, Tailwind and Shadcn

### Site:

https://newton-frontend.vercel.app/

### Features:

- Push based price update via websocket
- Search functionality
- Sort functionality
- CAD and USD support
